# crime-prediction-UK
---

## data sources
1. https://data.police.uk/data/
   - Use data from June 2020 - June 2022
   - All forces


## Setup
1. Create and activate a virtual environment
    - `python -m venv venv`
    - `venv/Scripts/activate`
2. Install packages
    - `pip install -r requirements.txt`
